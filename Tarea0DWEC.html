INMOBILIARIA

1-CREAR BASE DE DATOS

-Recogida de datos(entrada)

Se realiza la recogida de datos de cada vivienda que se va a almacenar en la base de datos(propietario,dirección,es alquiler o venta, precio…)

-Interacción hombre-máquina

Los datos obtenidos son guardados en variables

-Almacenamiento

Mediante un método, los datos recogidos se almacenan en distintas variables y dentro de su array correspondiente.

-Modificación de la base de datos

También será creado un metodo que modifica o elmina los datos almacenados

2-BÚSQUEDA DE VIVIENDA

-Recogida de datos(entrada)

Mediante varias preguntas al cliente(precio(mayor de y menor de),nº habitaciones, zona...), se recogen los datos necesarios para poder realizar la búsqueda en la base de datos de la vivienda que se desea.

-Interacción hombre-máquina

Los datos recogidos son transformados en variables

-cálculo

Se crea un método que busque en la base de datos la vivienda o viviendas que cumpla con los requisitos que busca el cliente.

-mostrar resultados

Se muestran por pantalla los resultados obtenidos.
